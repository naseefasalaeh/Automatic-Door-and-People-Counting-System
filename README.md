# 🚪 Automatic Door & People Counting System

![Arduino](https://img.shields.io/badge/Arduino-Uno-blue)
![Language](https://img.shields.io/badge/C++-Arduino-blue)

An Arduino-based automatic door system that automatically opens and closes the door while counting the number of people entering and leaving a room using dual ultrasonic sensors.

## ✨ Features

- 🚪 Automatic door opening and closing
- 👥 Real-time people counting
- 📡 Dual HC-SR04 ultrasonic sensors
- 📺 LCD I2C display
- ⚙️ Servo motor control
- ⏱️ Entry/Exit detection with direction recognition

---

## 🔧 Hardware

- Arduino UNO
- HC-SR04 Ultrasonic Sensor ×2
- SG90 Servo Motor
- LCD 16×2 I2C
- Breadboard
- Jumper Wires
- USB Cable

---

## 🛠 Software

- Arduino IDE
- C++
- LiquidCrystal_I2C Library
- Servo Library
